# TG-NPDCL-Commercial-Electricity-Consumption-Billing-Analysis-Oct-2025-

This project presents an interactive Power BI dashboard built using government-provided open data from the Telangana Open Data Portal (data.telangana.gov.in).
The dashboard analyzes commercial electricity consumption, billing efficiency, connected load, and regional demand patterns under TG NPDCL.
